Detailed Description

template<size_t NUM>
class AjaneRadioGroup< NUM >

A set of radio buttons (mutally exclusive buttons), e.g. for on/off, or low/mid/high, etc.

You can insert either the whole group into an ArudJAXPage at once, or - for more flexbile layouting - retrieve the individual buttons using() button, and insert them into the page as independent elements.
