void AjaneMutableSpan::setValue ( const char *  value,
bool  allowHTML = false 
)

Set the s content to the given value.

Parameters
valueNote: The string is not copied, so don't make this a temporary.
allowHTMLif true, you can set the content to any valid HTML string, which allows for much flexibility, but is not safe when value is untrusted user input. if false (the default), any "<" and "&" in value will be escaped, before rendering on the client, making the string plain but safe.
